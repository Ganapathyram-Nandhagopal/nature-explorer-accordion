# 🌿 Nature Explorer Accordion

A modern and interactive nature-themed accordion gallery built using pure HTML, CSS, and JavaScript. Featuring smooth panel expansion animations, beautiful landscape imagery, ambient mode, responsive design, and an elegant user experience.

![Preview](preview.png)

---

## ✨ Features

* 🌿 Nature-Inspired Design
* 🎭 Smooth Accordion Expansion Animations
* 🖼️ Stunning Landscape Galleries
* 🎨 Modern Minimal UI
* 🌙 Ambient Mode Toggle
* 🔀 Shuffle Gallery Feature
* ⬅️➡️ Previous & Next Navigation
* 📱 Fully Responsive Design
* ⚡ Lightweight & Fast

## ⚙️ Customization

### Change Images

Update image URLs in `index.html`:

```html
<div class="panel-bg"
     style="background-image:url('your-image-url.jpg')">
</div>
```

### Add New Categories

Duplicate a panel block and update:

```html
data-title="New Category"
data-accent="#4CAF50"
```
